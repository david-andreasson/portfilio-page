Examensarbete – AI-drivna integrationer

Förstudie inför kommande examensarbete där moderna integrationer mellan AI-tjänster, kryptoteknik och community-plattformar (Discord) utforskas. Målet är att bygga en demonstrationslösning som visar upp teknisk bredd och integrationstänk.

Projektmål

  • Bygga en backend i Java/Spring Boot som kopplar samman AI-API:er, kryptotjänster och Discord.
  • Demonstrera realtidsflöden där användare får rekommendationer, notifieringar och dashboards.
  • Fokusera på säkerhet, spårbarhet och hur data rör sig mellan externa tjänster.
  • Dokumentera lösningen så att den kan visas upp för potentiella arbetsgivare.

Planerad teknikstack

  • Backend: Java 21, Spring Boot 3, Spring Security
  • AI-integration: TBD
  • Crypto: TBD
  • Community: Discord-integration, möjligtvis Telegram
  • Data: TBD
  • Deploy:TBD - Möjligvis Docker och GitHub Actions

Arkitektur-idéer

Lösningen planeras som en mikrotjänst-inspirerad struktur där huvud-API:t fungerar som hubb och integrerar olika externa tjänster.
