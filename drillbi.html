Drillbi – Adaptiv quizplattform

En AI-stödd quizplattform som kan generera nya frågor utifrån ditt material och låter dig repetera svåra frågor oftare. Drillbi hjälper studenter att lära sig effektivare genom att automatiskt generera quizfrågor från text eller bilder, och genom att återinföra felaktigt besvarade frågor efter en stund för att befästa kunskapen.

Teknisk översikt

  • Tech stack: React 19 + TypeScript (frontend), Spring Boot 3.4 + Java 21 (backend)
  • State management: React Context API för global state, custom hooks för lokal state
  • Autentisering: OAuth2 (Google/GitHub) + JWT-tokens, Spring Security
  • Deployment: Docker-containers, CI/CD med GitHub Actions
  • Testning: JUnit 5 med JaCoCo-täckningsrapporter för backend

Arkitektur & integrationer

Drillbi är byggd med en modern mikroservice-arkitektur där frontend och backend är helt separerade. Systemet integrerar flera externa tjänster för att leverera en komplett lärplattform.

  • REST API: Backend exponerar ett RESTful API med versionshantering (/api/v2/)
  • OpenAI Integration: GPT-modeller för att generera quizfrågor och AI-förklaringar
  • Anthropic Claude: Alternativ AI-modell för frågegenerering (Claude 3 Haiku)
  • OCR med Tesseract: Extraherar text från uppladdade bilder för att skapa quiz
  • Stripe Payments: Hanterar prenumerationer och betalningar för premium-funktioner
  • MySQL Database: Persistent lagring av användare, kurser och frågor
  • H2 Database: In-memory databas för utveckling och testning
  • JPA & Hibernate: ORM för databashantering med automatisk schema-generering

Nyckelfunktioner

  • AI-genererade frågor: Skapa automatiskt quizfrågor från valfri text med OpenAI eller Claude
  • Bild-till-quiz: Ladda upp bilder av läroböcker eller anteckningar, OCR extraherar texten och AI genererar frågor
  • Adaptiv repetition: Felaktigt besvarade frågor läggs tillbaka i kön efter 10-15 frågor för effektiv inlärning
  • AI-förklaringar: Få detaljerade förklaringar på varje fråga och svar med hjälp av AI
  • Flerspråkigt stöd: Fullständigt stöd för svenska och engelska med i18next
  • Kurshantering: Skapa, redigera och organisera kurser med anpassade frågor
  • Frågeredigering: Manuellt redigera AI-genererade frågor eller skapa egna från grunden
  • Statistik & progress: Realtidsstatistik över rätta svar, felfrekvens och framsteg
  • Premium-funktioner: Stripe-integration för prenumerationer med utökade AI-funktioner

Lärdomar

Backend-utvecklingen av Drillbi har varit en djupdykning i modern Java-utveckling med Spring Boot. Projektet har gett omfattande erfarenhet av att bygga robusta REST APIs, hantera komplex affärslogik och integrera externa tjänster som OpenAI, Anthropic Claude och Stripe. Att arbeta med Spring Security för OAuth2-autentisering och JWT-tokens har gett solid förståelse för säker autentiseringshantering i produktionsmiljö.

Integration med flera AI-modeller (OpenAI GPT och Claude) krävde omfattande arbete med prompt-engineering, felhantering och parsing av JSON-svar från olika API:er. Att hantera OCR med Tesseract för bild-till-text-funktionalitet var också en lärorik utmaning, särskilt kring bildbehandling och textextraktion. Stripe-integrationen lärde mig hur man hanterar webhooks, betalningsflöden och prenumerationshantering på ett säkert sätt.
